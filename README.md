# Projeto Entrega Intermediária

Link da aplicação publicada: (https://integra-o-com-api-via-cep-para-pree-navy.vercel.app)

## Descrição
Aplicação web que consulta endereços automaticamente utilizando a API pública ViaCEP.

## Funcionalidades
- Consulta de CEP
- Retorno de logradouro, bairro, cidade e estado
- Integração com API REST pública
- Teste de integração automatizado

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Node.js
- ViaCEP

## Como executar
```bash
npm install
npm test
Entrega intermediária finalizada com integração ViaCEP.
